# My Portfolio Website
https://fuku-hiro112.github.io/my-website/

個人ポートフォリオサイトです。

## Overview

ソフトウェアエンジニアとしてのスキル・経歴・制作物をまとめたポートフォリオサイトです。

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Structure

```
my-website/
├── index.html    # メインページ
├── style.css     # スタイルシート
└── README.md
```

## Sections

| Section | Description |
|---------|-------------|
| About | 自己紹介・業務概要 |
| Skills | 技術スキル（言語・製品・ツール） |
| Experience | 職務経歴 |
| Works | 制作物（学生時代のゲーム開発等） |
| Certifications | 保有資格 |
| Strengths | 強み・特長 |

## Features

- レスポンシブデザイン対応（PC / タブレット / スマートフォン）
- スクロール連動のフェードインアニメーション
- 固定ナビゲーションバー（スクロール時に背景変化）

## Development

本ポートフォリオは Claude Code を活用して開発しています。
